Downloading and Running Scapy
To install Scapy on a Debian server, execute these commands:

```sh
$ sudo apt update
$ sudo apt install git -y
$ git clone https://github.com/secdev/scapy.git
$ cd scapy
$ sudo ./run_scapy
```
