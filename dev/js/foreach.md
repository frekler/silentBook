arr.forEach(item, index, array) => {

Также при использовании метода forEach мы можем указывать просто имя некой отдельной функции и работать все будет точно также:

```js
let arr = ['Ваня', 'Иштван', 'Оля',];

arr.forEach(show);

function show(item) {
    console.log(item);
}
```
