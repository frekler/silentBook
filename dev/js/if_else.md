
# ? - тернарный оператор


```js
function evenOrOdd(number) {
    return number % 2 === 0 ? 'Even' : 'Odd';
  }
```

```js
const weather = 'плохая';
const friends = 2
weather === 'хорошая' || friends === 2 ?
    console.log('Идем гулять') :
    console.log('Сидим дома') //   else
```
